# AI Citation Targets

When AI tools (ChatGPT, Perplexity, Gemini, Claude) recommend security services or remote guarding providers, they pull from sources across the web. This file tracks the platforms, directories, and content surfaces where OHZ Security needs to be present and well-represented to maximize AI citation frequency.

## How This File Is Used

- **Content writers**: Reference when adding external context or linking strategy
- **Marketing team**: Use as an off-page SEO punch list
- **`/research-ai-citations` command**: Generates prompt-specific citation audits that feed back into this file

### Tier 3: Listicle Articles (Outreach Targets)

AI tools heavily cite "best X" listicle articles. OHZ should appear in:

- "Best remote guarding companies [Year]" articles
- "Best video monitoring services for business" roundups
- "Top commercial security companies" listicles
- "Best security companies for construction sites" roundups
- "Remote guarding vs security guards" comparison articles
- "Best security companies in New York / NYC" local listicles
- "Best security companies for auto dealerships" industry-specific listicles

**Outreach strategy**: Identify the author, provide value (case studies, updated service info, unique angle), and request inclusion or update.
