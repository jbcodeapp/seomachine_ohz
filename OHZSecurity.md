## 🏢 Company Overview  
**OHZ Security (OHZ Remote Guarding)** is a U.S.-based security technology company specializing in AI-powered remote guarding and video surveillance solutions. It provides real-time protection for businesses and properties using a combination of human security agents and advanced AI systems.

- **Founded:** Around 2018  
- **Headquarters:** Brooklyn, New York, USA  
- **Focus:** Intelligent, remote security monitoring for commercial and industrial clients  

---

## 🔐 What OHZ Security Does  
OHZ delivers 24/7 remote security monitoring services that actively prevent crime instead of just recording incidents.

- Real-time video surveillance with live guards  
- AI-powered threat detection  
- Instant response and intervention  
- Remote access control and monitoring  

---

## ⚙️ Core Services  

### 1. Remote Guarding
- Live monitoring by trained security agents  
- Immediate response to suspicious activity  
- Continuous surveillance of properties  

### 2. AI-Powered Security Monitoring
- Smart detection (people, vehicles, objects)  
- Automated alerts and event tracking  
- Cloud-based video management system  

### 3. Event-Based Security
- Detects motion, intrusion, or unusual activity  
- Triggers real-time alerts and responses  

### 4. Virtual Doorman
- Remote access control for buildings  
- Visitor verification and entry management  

### 5. Active Deterrence
- Two-way audio warnings to intruders  
- Escalation to law enforcement if needed
